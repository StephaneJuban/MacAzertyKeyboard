# MacAzertyKeyboard
Reference for every (common) special characters shortcut on a Mac keyboard (Azerty)

---

| Name              | Symbol        | Shortcut                 |
| :---------------: |:-------------:| :-----------------------:|
| Backslash         | \\            | `⌥` + `⇧` + `/`          |
| Euro              | €             | `⌥` + `$`                |
| Pipe              | \|            | `⌥` + `⇧` + `l`          |
| Tilde             | ~             | `⌥` + `n`                |

---

Some shortcut are not supported natively via shortcut

| Name              | Symbol        |
| :---------------: |:-------------:|
| Power             | ²             |
